This guide explains the structure, customization, and deployment of the EduConsultancy landing page template.

  • index.html: Main landing page file
  • contact.php: Example backend handler for contact form
  • assets/: Contains CSS, fonts, images, JS, and downloadable resources
  • Documentation/documentation.html: This documentation file

2. Customization

  • Text & Content: Edit index.html to update all visible text, headings, and testimonials.
  • Images: Replace images in assets/images/ with your own. Update src attributes in index.html if filenames change.
  • Styles: Modify assets/css/style.css for colors, fonts, and layout.
  • Resources: Add your downloadable files to assets/resources/ and update links in index.html.
  • Social Links: Update URLs in the footer of index.html.

3. Forms & Backend

  • The contact form submits to contact.php. Implement your own backend logic to process and send emails.
  • For newsletter or resource downloads, connect to your preferred email marketing or backend service.

4. Deployment

  • Zip the entire project folder for distribution or upload.
  • Upload all files and folders to your web server, keeping the structure unchanged.
  • Ensure index.html is in the root directory.
  • Place downloadable resources in assets/resources/ if needed.
