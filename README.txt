EduConsultancy Landing Page
==========================

Project Description
-------------------
A professional, responsive landing page template for education consultancy businesses. Designed to help agencies showcase their services, attract students, and collect leads for study abroad, career counseling, and visa assistance.

Features
--------
- Modern, mobile-friendly design
- About, Services, and Testimonials sections
- Downloadable resources (guides, checklists, SOP templates)
- Newsletter subscription and contact forms
- Social media integration

Folder Structure
----------------
- index.html                Main landing page
- contact.php               (Sample backend handler for contact form)
- README.txt                Project overview and instructions
- assets/
    css/
      style.css             Main stylesheet
    fonts/                  (Custom or Google Fonts)
    images/                 Images for team, testimonials, etc.
    js/
      main.js               JavaScript (empty by default)
    resources/              Downloadable files (PDF, DOCX, etc.)
- Documentation/
    documentation.html      Project documentation

Getting Started
---------------
1. Open `index.html` in your browser to preview the landing page.
2. Replace images in `assets/images/` with your own.
3. Edit text and links in `index.html` to match your brand and services.
4. Add downloadable resources to `assets/resources/` and update links.
5. Implement backend logic in `contact.php` for form handling.

Customization
-------------
- Change colors, fonts, or layout in `assets/css/style.css`.
- Update social media links in the footer of `index.html`.
- Add or remove sections as needed.
- Edit or insert your personal logo.

Deployment
----------
1. Zip the entire project folder.
2. Upload to your web hosting or server, keeping the folder structure intact.
3. Ensure `index.html` is at the root for best results.
